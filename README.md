Этот проект содержит автоматизированные e2e-тесты, написанные с использованием Cypress.
Тесты покрывают основные пользовательские сценарии, такие как авторизация, создание сущностей и изменение пароля.

**1. Установка зависимостей**

Перед началом убедитесь, что установлен Node.js.

Установите зависимости проекта:

`npm install`

**2. Подготовка тестовых данных (для теста CreateBox)**

Перед запуском тестов необходимо создать файл с тестовыми данными:

- Создать файл cypress.env.json на основе users.example.json

- Добавить реальные учетные данные (логин/пароль)

- Сохранить файл

**3. Запуск тестов**

В проекте предусмотрены готовые скрипты для запуска тестов:

- Запуск тестов в Cypress в интерактивном режиме (GUI)
  `npm run cy:open`

- Запуск всех тестов в режиме headless
  `npm run cy:run`

- Запуск feature-файла (BDD)
  `npm run cy:feature`

- Запуск теста создания коробки (CreateBox)
  `npm run cy:spec1`

- Запуск теста изменения пароля (UI)
  `npm run cy:spec2`

- Запуск теста изменения пароля (API + UI)
  `npm run cy:spec3`
